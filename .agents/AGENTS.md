# Project Rules

## GitHub Push on Completion
- Whenever you finish editing or resolving tasks in this project, and the user explicitly confirms that the task is completed/approved, you must offer to push the changes to GitHub.
- If confirmed by the user, automatically run the following commands to commit and push the changes:
  1. `git add .`
  2. `git commit -m "<descriptive message of what was changed>"`
  3. `git push origin main`
